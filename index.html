Drink Mistério

Máquina automática com sensores ultrassônicos — Projeto de Automação

Autores: João Rafael, Kezia Aguiar, Yasmin Karolina, Thayna Verçosa, Larissa Morais, Diogo Alcelino

Descrição do Projeto

Esse projeto é importante porque a automação está cada vez mais presente em dispositivos do dia a dia, especialmente em sistemas que priorizam praticidade, higiene e redução de contato físico. O protótipo demonstra como sensores, microcontroladores e atuadores podem ser integrados para criar soluções práticas e eficientes.

O objetivo é desenvolver uma máquina automática capaz de liberar líquido usando sensores ultrassônicos. Ao detectar um copo próximo, o sistema aciona LEDs e duas bombas de água, liberando o líquido pelos caninhos.

É um projeto voltado para automação, eletrônica embarcada e sistemas de controle, semelhantes a máquinas industriais, vending machines e sistemas de higienização sem toque.

  • Ponte H L9110S (Driver) – Código: L9110S
  • 2 Baterias 3.7V – Código: 18650
  • 2 LEDs + 2 Resistores 220Ω – Código: 220R
  • Jumpers diversos

Passo a Passo

Código Completo (Arduino)


#define IN1 3
#define IN2 4
#define LED1 7

#define TRIG1 8
#define ECHO1 9

#define IB1 5
#define IB2 6
#define LED2 10
#define TRIG2 11
#define ECHO2 12

long duracao;
int distancia;

void setup() {
  pinMode(IN1, OUTPUT);
  pinMode(IN2, OUTPUT);
  pinMode(LED1, OUTPUT);

  pinMode(TRIG1, OUTPUT);
  pinMode(ECHO1, INPUT);

  digitalWrite(IN1, LOW);
  digitalWrite(IN2, LOW);
  digitalWrite(LED1, LOW);

  pinMode(IB1, OUTPUT);
  pinMode(IB2, OUTPUT);
  pinMode(LED2, OUTPUT);

  pinMode(TRIG2, OUTPUT);
  pinMode(ECHO2, INPUT);

  digitalWrite(IB1, LOW);
  digitalWrite(IB2, LOW);
  digitalWrite(LED2, LOW);
}

void loop() {
  digitalWrite(TRIG1, LOW);
  delayMicroseconds(2);
  digitalWrite(TRIG1, HIGH);
  delayMicroseconds(10);
  digitalWrite(TRIG1, LOW);

  duracao = pulseIn(ECHO1, HIGH);
  distancia = duracao * 0.034 / 2;

  if (distancia > 0 && distancia < 10) {
    digitalWrite(LED1, HIGH);
    digitalWrite(IN1, HIGH);
    digitalWrite(IN2, LOW);
  } else {
    digitalWrite(LED1, LOW);
    digitalWrite(IN1, LOW);
    digitalWrite(IN2, LOW);
  }

  digitalWrite(TRIG2, LOW);
  delayMicroseconds(2);
  digitalWrite(TRIG2, HIGH);
  delayMicroseconds(10);
  digitalWrite(TRIG2, LOW);

  duracao = pulseIn(ECHO2, HIGH);
  distancia = duracao * 0.034 / 2;

  if (distancia > 0 && distancia < 10) {
    digitalWrite(LED2, HIGH);
    digitalWrite(IB1, HIGH);
    digitalWrite(IB2, LOW);
  } else {
    digitalWrite(LED2, LOW);
    digitalWrite(IB1, LOW);
    digitalWrite(IB2, LOW);
  }

  delay(50);
}

Problemas enfrentados e lições aprendidas

Tivemos problemas como ponte H queimada, motor queimado e vazamentos de água. Também enfrentamos dificuldades com o código e o circuito não respondendo corretamente. Com testes e pesquisas, conseguimos resolver tudo.

Aprendemos a importância de testar cada etapa, revisar o circuito e entender melhor alimentação e segurança dos componentes.
